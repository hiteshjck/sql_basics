<h1>Description</h1>
The ISNULL() function returns a specified value if the expression is NULL.

If the expression is NOT NULL, this function returns the expression.
<h1>Query</h1>
SELECT ISNULL('Hello', 'W3Schools.com');

SELECT ISNULL(NULL, 500);
